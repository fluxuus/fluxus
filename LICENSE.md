**❌ Key system won't complete / Linkvertise loop**
→ Clear your browser cache, disable ad-blockers, try a different browser, or use a VPN. The key system sometimes gets stuck — wait 5 minutes and retry.

**❌ "Fluxus is outdated" error after injection**
→ Roblox just updated. The Fluxus team typically releases a patch within 24-48 hours. Check the Discord for updates.

**❌ Injection fails / executor crashes**
→ Run Fluxus as Administrator. Ensure your antivirus has excluded the Fluxus folder. Reinstall if the issue persists.

**❌ Mobile version won't inject**
→ Ensure Roblox is fully updated. On Android, check that Fluxus has overlay permission enabled. On iOS, your sideload certificate may have expired — reinstall through AltStore.

**❌ Script executes but nothing happens**
→ Some scripts require specific game versions or UI libraries. Test with a known working script (try the built-in Script Hub first).

**❌ Windows SmartScreen blocks the installer**
→ Click "More Info" then "Run Anyway." This is a standard false positive for any Roblox executor due to the injection method.

**❌ Fluxus closes immediately after opening**
→ Your antivirus likely quarantined it. Restore the file from quarantine and add the entire Fluxus folder to exclusions before reopening.
